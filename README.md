# Funstore Library Management System

A fully functional, browser-based Library Management System built with pure HTML5, CSS3, and JavaScript — no external frameworks or backend required.

## Features

- **Role-based access** for 3 user types: Administrator, Manager, and Librarian
- **Book issue/return tracking** with automated overdue detection
- **Real-time inventory** across a catalogue of 111 B.Sc. CS books
- **50 student records** managed with full CRUD functionality
- **100% client-side data persistence** using localStorage
- **Interactive dashboard** with live search, advanced filters, and real-time statistics
- Responsive UI with a clean indigo/purple design

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- localStorage (browser-based storage)

## How to Run

1. Download or clone this repository
2. Open `index.html` in any modern web browser
3. No installation or setup required
